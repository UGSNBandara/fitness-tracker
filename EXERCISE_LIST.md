# Exercise List for Image Generation

## Week 1 - Beginner Level

1. **Push-ups** (ID: 1)
   - Muscle Group: Chest
   - Target: 8 reps × 3 sets
   - Description: Standard push-up position with hands shoulder-width apart

2. **Squats** (ID: 2)
   - Muscle Group: Legs
   - Target: 12 reps × 3 sets
   - Description: Basic bodyweight squat with proper form

3. **Plank Hold** (ID: 3)
   - Muscle Group: Core
   - Target: 20 seconds × 3 sets
   - Description: Forearm plank position maintaining straight body

4. **Jumping Jacks** (ID: 4)
   - Muscle Group: Full Body
   - Target: 15 reps × 3 sets
   - Description: Classic cardio warm-up exercise

## Image Requirements

- **Format**: PNG or JPG
- **Dimensions**: 400x280px (landscape orientation recommended)
- **Style**: Clear demonstration of exercise form
- **Background**: Clean, minimal background
- **Naming Convention**: `exercise_[id].png` (e.g., `exercise_1.png` for Push-ups)

## Storage Location

Images should be placed in: `assets/images/exercises/`

Don't forget to add the assets path to `pubspec.yaml`:
```yaml
flutter:
  assets:
    - assets/images/exercises/
```
